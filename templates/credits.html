GroundScratch uses these technologies to make it up and running:

Flask, for the web framework

Flask-Login, for the login back-end infrastucture

Flask-Cors, for the CORS (Cross-Origin Resource Sharing) implementation

Flask-Migrate, for the migrate database functionnality

Polaris, the orginal base code of GroundScratch.

Thanks to all of these guys that made these technologies for powering GroundScratch!
